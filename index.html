Fibonacci Spiral Generator Add-on Portfolio

Overview

The Fibonacci Spiral Generator is a custom Blender add-on designed to dynamically generate Fibonacci spirals accompanied by cubes, offering both visual and mathematical appeal. This add-on highlights the power of Python scripting within Blender, enabling users to combine geometry, artistic design, and mathematical precision in their projects.

Key Features

  • Dynamic Fibonacci Spiral Creation: Generates Fibonacci spirals with customizable segment counts (1-20).
  • User-Friendly Interface: Accessible through the Sidebar in the 3D Viewport under the Fibonacci tab.
  • Custom Geometry Integration: Spirals are generated using Blender's curve objects, providing editable 3D curves.
  • Automated Workflow: Clears the scene and resets the environment for seamless Fibonacci spiral creation.

Blender Version Compatibility

The Fibonacci Spiral Generator is designed to leverage the latest Blender Python API features while maintaining compatibility across a wide range of versions:

  • Minimum Recommended Version: Blender 2.8
  • Fully Tested Versions: Blender 2.8, 2.9, 3.x, and 4.3
  • Current Target Version: Blender 4.3

Instructions for Installing, Running, and Uninstalling the Fibonacci Spiral Generator Add-on (Using a ZIP File)

Installing the Add-on

  1. Prepare the ZIP File:
    • Ensure the add-on script (fibonacci_spiral_generator.py) is saved inside a ZIP file named fibonacci_spiral_generator.zip.
  2. Open Blender:
    • Launch Blender and go to Edit > Preferences (Windows/Linux) or Blender > Preferences (macOS).
  3. Go to the Add-ons Tab:
    • In the Preferences window, click on the Add-ons tab on the left.
  4. Install the Add-on from the ZIP File:
    • Click the Install... button in the top-right corner of the Add-ons tab.
    • In the file browser that opens, locate and select the fibonacci_spiral_generator.zip file.
    • Click Install Add-on.
  5. Enable the Add-on:
    • After installation, search for the add-on by typing Fibonacci in the search bar.
    • Check the box next to Fibonacci Spiral Generator to enable it.

Running the Add-on

  1. Access the Add-on in the 3D Viewport:
    • Open the 3D Viewport and press the N key to open the Sidebar.
    • Navigate to the Fibonacci tab in the Sidebar.
  2. Use the Panel:
    • In the Fibonacci Spiral Generator panel:
    • Adjust the number of Fibonacci segments using the slider labeled 1 - 20 Segments.
    • Click the Generate Fibonacci Spirals & Cubes button to create the spiral in the scene.

Uninstalling the Add-on

  1. Open Preferences:
    • Go to Edit > Preferences (Windows/Linux) or Blender > Preferences (macOS).
  2. Navigate to the Add-ons Tab:
    • In the Preferences window, click on the Add-ons tab.
  3. Search for the Add-on:
    • Type Fibonacci in the search bar at the top.
  4. Uninstall the Add-on:
    • Click the small arrow next to the add-on name to expand its details.
    • Click the Remove button to uninstall the add-on completely.

Important Notes:

  • ZIP File Structure: Ensure the ZIP file contains only the .py script file and no additional folders. Blender requires this structure for proper installation.
  • Add-on Updates: If updating the add-on, uninstall the existing version first and then reinstall using the new ZIP file.
  • Add-on Compatibility: The Fibonacci Spiral Generator is compatible with Blender versions 2.8 and above.
